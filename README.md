# Event App (Android)_Ho Van Tiet

## Introduction
Event App is a simple Android application designed to manage and display event information.

The application allows users to perform basic CRUD operations (Create, Read, Update, Delete) on events.

This project was developed as part of my learning journey in Android development.

---

## Features
- Authentication: Register, Login, Logout  
- Home screen: Overview of total events, past events, and upcoming events within the next 24 hours  
- Event management: View, add, update, delete, and track events  
- Additional features: Add notes, notifications for user actions, edit profile, and more  
- Local data storage using SQLite  

---

## Technologies Used
- Java (Android)
- XML (UI Design)
- SQLite
- Android Studio

# Ứng dụng Sự kiện (Android)_Hồ Văn Tiết

## Giới thiệu
Ứng dụng Sự kiện là một ứng dụng Android đơn giản được thiết kế để quản lý và hiển thị thông tin sự kiện.

Ứng dụng cho phép người dùng thực hiện các thao tác CRUD cơ bản (Tạo, Đọc, Cập nhật, Xóa) trên các sự kiện.

Dự án này được phát triển như một phần trong hành trình học tập của tôi về lập trình Android.
